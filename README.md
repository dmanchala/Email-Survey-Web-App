# Pollstery

A practice web application to send yes-no email surveys to customers. Built as part of a Udemy course.

Frameworks:
* React
* Redux
* Node
* Express
* Materialize

Services:
* Heroku
* Google OAuth API
* SendGrid
* Stripe
* mLab
